## 10 - WEATHER

指定世界上的天气设置。所有的数值都是以ticks来测量的。

### 可选选项

| 变量 | 描述 | 缺省值 |
| --- | --- | --- |
| MaxSunnyTicks |  | 180000 |
| MinSunnyTicks |  | 12000 |
| MaxRainTicks |  | 24000 |
| MinRainTicks |  | 12000 |
| MaxThunderStormTicks |  | 15600 |
| MinThunderStormTicks |  | 3600 |
